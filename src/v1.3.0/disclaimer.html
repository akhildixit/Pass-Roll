Privacy Policy

What we collect

We do not collect, store or share any information from the Pass-O user.
The Chrome Extension installed on Pass-O user's machine temporarily reads passwords that Pass-O user types on a web page on which she has enabled Pass-O. Chrome Extension uses these passwords for the sole purpose of providing a personalized user interface to rotate the passwords. Passwords are never sent out of Pass-O user's machine in any form by our Chrome Extension. Chrome Extension does not save these passwords in any form, and they are deleted permanently from the transient local memory of Pass-O user's machine immediately after the web page reloads or new web page loads or the browser tab or browser window is closed.
