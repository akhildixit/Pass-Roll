Instructions of use

With Pass-O, you can rotate your existing passwords to make them more complex. Every time you enter your password, Pass-O will create a circular user interface where you can easily change the starting point of your password to rotate it.

  1. Visit options page

  2. In the text-box, type a keyword in the URL of the web-site on which you want to enable Pass-O. Host name of the web-site should contain whatever you type in the text-box. E.g. Typing 'xyz' in the text-box will enable Pass-O on web-sites like www.xyz.com, www.vwxyz.net, http://xyz123.in etc.

  3. Click on 'Add website' button

  4. Now whenever you type a password on that web-site, Pass-O will come up with an easy to use interface where you can change the starting point of your password.
